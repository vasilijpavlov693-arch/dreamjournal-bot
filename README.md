# Dream Journal Bot 🎙️

Telegram бот для ведения дневника снов с распознаванием голоса.

## Возможности
- 🎤 Расшифровка голосовых сообщений в текст
- ✨ Анализ и красивое описание снов
- 🖼️ Генерация иллюстраций к снам

## Локальный запуск

1. Клонировать репозиторий
2. Создать виртуальное окружение: `python -m venv venv`
3. Активировать: `venv\Scripts\activate` (Windows)
4. Установить зависимости: `pip install -r requirements.txt`
5. Создать `.env` из `.env.example` и заполнить токены
6. Запустить: `python bot.py`
